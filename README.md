## Arquitetura
Arquitetura construida com preocupações de separação de responsabilidades, seguindo as boas práticas de SOLID e Clean Code.

- Hexagonal Architecture
- Domain Driven Design
- Domain Events
- Domain Notification
- Domain Validations
- CQRS
- Retry Pattern
- Unit of Work
- Repository
